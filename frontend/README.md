# Veolve — Coming Soon Landing Page

This repository contains a production-ready static "coming soon" landing page for Veolve.
It is responsive, accessible, and ready to deploy to Hostinger (or any static hosting).

## Features
- Modern hero design inspired by provided reference image
- CTA email input (client-side validation)
- Social icons (placeholders)
- Dockerfile to serve via Nginx
- Simple local dev script and smoke test

## Local development
Requirements:
- Node >= 16 (for `npx http-server`) — optional

Run locally:
```bash
git clone <repo>
cd veolve-coming-soon
./run.sh
# then open http://localhost:8080
